#算法思路： 
在x,y以及时间t的三维空间内A*路径规划，检索（x+1,0,t+1）,（x-1,0,t+1）,（0,y+1,t+1）,（0,y-1,t+1）四个栅格。
1. 对于（x,y,t+1）有障碍物的格子不去考虑。
2. 同时考虑到两个智能体不能同时走在同一路径下，否则会碰撞，采取以下方法：
   若（x,y）在t+1时刻被agentN占领，则agentN在当前时间t所在的格子也不检索
